# 2D-medical-images-to-3D-using-VTK-and-python
## This Application Support:
    - Loading a DICOM file​
    - surface rendering (while allowing the user to adjust iso value)​
    - Ray Casting Rendering​
## Tools Used:
    - Python
    - VTK
    - PyQt5
